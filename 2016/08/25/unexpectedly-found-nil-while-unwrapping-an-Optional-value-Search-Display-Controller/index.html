unexpectedly found nil while unwrapping an Optional value(Search Display Controller)

原因是Search Display Controller启动后会新创建一个tableView,而我们的Cell的引用不能直接用此tableView reuse cell identifier,

必须要改为self.tableView … 这尼玛是什么原因啊！

Swift还是不错的，可以写出简洁的代码!
