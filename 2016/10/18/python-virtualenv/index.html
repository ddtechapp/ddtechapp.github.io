python virtualenv

virtualenv is a tool to create isolated Python environments.

virtualenv是一个python开发工具，按照官方文档来说，这个工具是创建一个不依赖现有电脑环境的独立开发环境。

安装:

sudo pip install virtualenv

创建开发目录:

virtualenv v-env

等待一段时间后，就会看到一个名字为v-env的目录，进入目录里面有三个文件夹: bin, include, lib

bin: 包含python的一些命令工具，python, pip, easy_install等等
include: python的头文件
lib: python的一些lib，这个文件有时间我也会读一读代码的

按照上述文件的内容，我们基本可以知道这里面包含了一个小的python开发环境，按照官方文档的提示，使用

source ./bin/activate

就可以开启开发环境, 使用全局命令deactivate又可以返回现有的环境。

happy hacking, done!
