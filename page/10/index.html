跑步，阅读，技术以及其他

似乎没到一个年龄段，比如十岁，二十岁，三十岁，四十岁，都会出现一个自己明显不太想承认，但是不得不去面对的状况，年龄危机，还好我终于度过了，无论是心态上，还是行为上，以后再去谈这个问题。

晚上跑完5KM，其实还有余力，可以跑到8KM，双腿这个时候不自觉放弃了，肺也不疼，身体也没有累到极限，就是自然而然的停下来了，停下来就停下来，路过ZOO COFFEE，好像进去喝一杯，现在又没有无糖的咖啡，我也不喜欢喝咖啡，只是想去坐一坐，休息一下。可一想到身上汗涔涔的，也只好作罢，走路回家。会上被一个健身的女的拉着添加了一下微信，我实在是烦腻这些事情，处于礼貌，加一个微信吧。美女是个美女，这和我有什么关系呢？我又懒得掏钱去做健身，我还要减掉10斤的体重才会考虑是否要掏钱报班。

最近几年，新兴中产阶级的名词不绝于耳，搞得好多人似乎都觉得自己是中产阶级，跑步，也就不自觉成为了一种信仰。跑步装备简单，门槛很低，一双跑鞋，迈开双腿就可以跑了，而且格调不算低，也能减肥，自然风靡全国。我不是中产阶级，我连房子都没有，只是看了春上村树的那本书就开始跑步了，断断续续跑了几个月。去珠海的那段时间跑的比较多，后面中断了，现在又捡起来了，五月份跑的很多的，跑了接近110KM，真是成绩斐然啊！后面越来越少了，第一是自己变得有点懒惰，还有就是纯粹没时间，再则家里发生了很多事情，让人无暇顾及。还好体重维持在148左右，还是比较开心的。

今天又去跑了5KM，争取这个月跑到80KM，体重降低到142吧。

说来好笑，我只是很仔细的读完了村上的那本跑步的书和《挪威的森林》，其他的书拿起来就放下了，有时候觉得自己纯粹就是个功利主义者，看书也非常功力，闲书从不多看，还是要把村上补充起来。昨天读完刘震云的《一句顶一万句》，文字还是非常多的，读了快一周，终于在昨晚看完，还是蛮兴奋的。这种兴奋不是指，内容很兴奋，而是指，阅读效率很高，自己很满意，就像三体读完那样的满足。很多时候惆怅不是因为自己想明白了，而是自己没有想明白事情。而阅读可以作为一种解药，去解答那些没有想明白的事情。《一句顶一万句》就给了我很多启发，比如日子要向后过，比如人生的浅薄，比如命运的无常。我们都是被生活推着走的蚍蜉，哪里是命运的强者，折腾生活更多的是满足自己内心的需求罢了。想明白这些，对待事情也就能够比较坦然了，生活本身不会给我们什么意义，而是我们赋予了生活太多的意义。

好好读书，好好生活。

前几天很想联系一下一个读书很好的同学，老是微信搜索出来后，等了半天又关掉了，已经过去十年了，还会有联系么？我已经不是以前的我了，他会是以前的他么？当然，我仅仅想问一下国外的生活怎么样，或许有一天我也会去国外生活，可我还是不想去麻烦别人了。这有点像《一句顶一万句》中的情节一样，说的再多，还不如一句话，说一句话，还不如不说。生活本该如此，随机而无常，我们自己花自飘零水自流的活着吧。

公司的事情实在是很无聊，开了个有趣的项目，我一个人写python后台，越写越有趣，仅仅是自high也是不错的。下午骑mobike在路上，想着生活，想着日子，我依旧还是热爱技术的，对技术的那种热情从未被浇灭，不要压抑自己，不要压抑自己。

从前面健身说起，健身推广的女的说我说话为什么紧张，当然不是和她长得漂亮有关，而是我不感兴趣的东西，都会显得局促，想尽早脱身，而变得说话紧张，我为什么会变得紧张呢？需要改掉这个毛病，这是长期的习惯吧，就像身上的肉一样，要慢慢改。

肉要一公里一公里的减掉

饭要一口一口的吃

口语要一句一句的联系

书要一本一本的看

日子要一天一天的过

相信未来~~~

技术功利主义

被儿子吵醒了，也无心睡眠，早起给家人做南瓜粥，然后躺沙发上刷手机，老婆床上刷手机，儿子床边看图画书，我沙发上刷手机，难得的一家三口和谐相处啊!

手机刷的实在无聊，边打开显示器看昨天的swift，实在不太明白泛型里面的Generator，看了几次都清晰，想放弃，觉得这个玩意儿目前用处不大。放弃不看，又捡起来看，非要搞懂为止，其实这个技术没那么难。不想搞懂一个东西的原因无非是，搞懂这个玩意儿能给我带来多少钱？

技术功利主义非常盛行，这也不能够怪谁，人是一种经济动物，不能够脱离社会母体，哪里挣钱往哪里奔，iOS挣钱学iOS，听说大数据挣钱，一股脑的往里面钻，天下熙熙攘攘皆为利来。为利的越多，乐趣就越少，乐趣少了，就越来越浮躁，不愿意花心思在本事的修炼上面，东搞搞，西弄弄，时间也就蹉跎了。

我目前最佩服的程序员，有四个，一个是linus，创造了linux, git,一个是春哥，写的一首好c代码，开源软件是openresty,一个是onevcat,iOS写的非常棒, 最后一个是matz，创造了ruby。成年人的生活本身不易，如果有自己的追求，这种追求短期内无法给自己代码多大的利益，我们就更应该保护起来，因为所爱，才能做得更好，才能在做的过程中觉得自己是一个有趣的人。

func calculateStatistics(scores: [Int]) -> (min: Int, max: Int, sum: Int) {

var max = scores[0]
var sum = 0

    if score < min {
        min = score
    } else if score > max {
        max = score
    }
    sum += score
}

return (min, max, sum)

}
