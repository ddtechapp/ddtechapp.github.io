技术功利主义

被儿子吵醒了，也无心睡眠，早起给家人做南瓜粥，然后躺沙发上刷手机，老婆床上刷手机，儿子床边看图画书，我沙发上刷手机，难得的一家三口和谐相处啊!

手机刷的实在无聊，边打开显示器看昨天的swift，实在不太明白泛型里面的Generator，看了几次都清晰，想放弃，觉得这个玩意儿目前用处不大。放弃不看，又捡起来看，非要搞懂为止，其实这个技术没那么难。不想搞懂一个东西的原因无非是，搞懂这个玩意儿能给我带来多少钱？

技术功利主义非常盛行，这也不能够怪谁，人是一种经济动物，不能够脱离社会母体，哪里挣钱往哪里奔，iOS挣钱学iOS，听说大数据挣钱，一股脑的往里面钻，天下熙熙攘攘皆为利来。为利的越多，乐趣就越少，乐趣少了，就越来越浮躁，不愿意花心思在本事的修炼上面，东搞搞，西弄弄，时间也就蹉跎了。

我目前最佩服的程序员，有四个，一个是linus，创造了linux, git,一个是春哥，写的一首好c代码，开源软件是openresty,一个是onevcat,iOS写的非常棒, 最后一个是matz，创造了ruby。成年人的生活本身不易，如果有自己的追求，这种追求短期内无法给自己代码多大的利益，我们就更应该保护起来，因为所爱，才能做得更好，才能在做的过程中觉得自己是一个有趣的人。

func calculateStatistics(scores: [Int]) -> (min: Int, max: Int, sum: Int) {
var min = scores[0]
var max = scores[0]
var sum = 0

for score in scores {
    if score < min {
        min = score
    } else if score > max {
        max = score
    }
    sum += score
}

return (min, max, sum)

}

白驹过隙

又是难的的周末，下班之后就直接装好书包走人，工作都留给下周吧!

信步回家，听着罗辑思维，挺着腰杆的走路-我实在练习走路的姿态，前三十年走路的确佝偻这肩膀，没有气势，也没有仪态，非常难看，要改正。屁股还是有点疼，不能够任性的骑车子回家，还是蛮遗憾的。周日去参加餐饮协会的举办的活动，还是有机会骑车的。

到家，儿子非常乖的喊叫dad，然后开心的跑开自己玩自己的。他有他的乐趣，人生真的很奇妙，以前觉得自己可能会单身很久，没想到不到三年，老婆，儿子都有了。我也快30岁了。不得不慨叹时光的蹉跎，蹉跎六年的工作，我没有赚很多钱，没有创业发达，只是老老实实工作，生活，每天回到家，有媳妇，有娃，有吵闹，有怨言，有烦恼，我想这就是人间的烟火吧！

翻看知乎，看到一题很有趣，如何度过自己的一生，才会让自己觉得这一生没有白驹过隙？

大家答的到很达观，可生活中又有谁能够少年不为红粉所困，青年不为名利而恼，老年不为儿女而怒呢？

不过生命实在太短暂了，真的要认真过好每一天，每一分每一秒…

浅薄

搜索TestFlight的时候搜索到一个知名iOS开发者的博客,读了2015年的总结，很感慨，博客让我看到的少有的淡然，他应该比我大一岁，清华大学毕业，目前在Line做一名iOS工程师。起点如此之高，按照普通人的思维，应该去创业啊，而不是做什么修行，写什么代码。浮躁的不是社会，而是社会上的人。

我也快到三十岁了，昨天读到一篇文章说，三十岁就是一个坎，人会有一个心里负担，看到周边的人创业，买房，人就不自觉的焦虑起来。这是心理病，得治疗。

当看到stay hungry stay foolish的时候，我感到很羞愧，多年前我也是那这个作为自己的座右铭，但我真的践行了么？我还不是终日无所事事，蹉跎时光。

无知绝对不是阻碍进步的原因，浅薄才是。以为自己懂得很多，以为自己很厉害。要努力去尝试，要努力去吸收各种知识，要谦虚，要勤勉，不要做一个浅薄到自己都讨厌的人。
