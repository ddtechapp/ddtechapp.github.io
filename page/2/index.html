talk is cheap

准备在公司写点什么，放假放的过于仓促，其实也不知道写一些什么。年终奖又一次没有发，其实我倒无所谓，就像承诺的奖金也不知道什么时候发，那虚无缥缈的四年的年终奖一起发也随着风飘散了吧。我其实并不在意这些东西，我只是在意一些承诺，一些言之凿凿的承诺罢了。

Fenng说的对，与其纠结这些，不如思考自己的将要要做什么，行业不会对真正有抱负的人有所亏欠。

so, talk is cheap, I need to build something.

17年我会做几个外包性质的项目，多为创业项目的0.1版本，然后做点自己的业余SAAS项目，积累一些自己的东西。我终究不会在一个公司呆很久，也不会去一个所谓的大公司甩尾。我讨厌大公司，只要有足够的利润养活自己，我宁愿做自己的事情，做自己喜欢的东西，也不愿为钱而一味的加班，透支自己的生命。

我需要一些时间来完成自己心中的想法，be patient!

平常心和持续去做，坚韧不拔的精神。

2017-01-18

Why PHP?

自从干完自己的官网之后，我就开始学习PHP开发，以前看不起PHP，觉得不就是个脚本语言，而且和html,javascript纠结在一起，还有令人不安的$符号。这是一个让有洁癖的人很不舒服的语言。

去年一年，我花了太多的时间学习Rails开发，学习如何适应这个框架的各种最佳实践和黑魔法，然而，我到目前为止都很难说掌握Rails开发，谢谢简单的hello world代码是没什么问题的，要干一个复杂的网站，就歇菜了。我没有办法很好的处理Rails，很好的用Ruby语言，很好的去部署Rails代码。

今天再看PHP MYSQL JAVASCRIPT那本书，第一句话就击中了我的内心，PHP很简洁，不像Rails那样那么复杂。

写了这么多年的代码，难道这种简洁不是我所需要的吗？

神马面条一样的代码，神马不可维护的代码，神马入门简单的代码。

很多时候，我们需要的是把事情搞完。

这个世界上90%的网站不需要复杂的部署，复杂的逻辑，复杂的页面。

我们需要为这个90%的客户服务，而不是另外那10%。
