keep going

过年在家有时间就读书，年前买了本村上春树的新书《我的职业是小说家》，这是村上的一本散文体自传。

去年差不多读了四五本村上春树的小说，今年继续把剩下所有的小说读完。

这本书如其说是一个作家的日常，不如说是一本克制的方法论的书，如何在年轻的时候打拼，如何以作家这
个身份生活，如何写短篇小说，如何写长篇小说。

写作和写代码类似，只不过写作是普通人难以企及的东西。

以前看过一本书讲如何刻意训练自己，村上春树也提到过这一点，当然他提出的是更加具体的事情–如何
稳定的产出，比如如何每天坚持不懈的写十页400字的小说，每天要跑步一个小时，在写作的时候，绝不会
干其他的事情分心，宁愿吃老本，也不会去为了赚钱而做一些快餐式的写作，更加坚持的做一件事情才能做好。

我觉得这些都很有启发性，以前对自己的决定和想法不是由太多的自信，现在觉得无论什么决定都是我深入
思考过的，只要坚持做下去，一定会有成绩的，关键还是有规律的坚持。

happy new year

过年没有回家，在深圳待着也不错，做饭，卤菜，包饺子，写春联，做西餐，带孩子…

我不喜欢过年，就像很长一段时间我都不喜欢家里人一样，过年时很无趣的事情，说一些令人不爽的场面话，到处跑一跑。年轻单身的时候倒无所谓，结婚生子有了自己的家庭后，却不觉非常乏味。能回避就回避，过年在家好好休假，在深圳的出租房里面待着，做饭，带小孩，休息，也不错。

朋友说家里几天，非常不爽，在外面久了，就是会有这种情绪在里面，没办法。

在外面过完一个年，才觉得自己终于有一个完整的家，不需要依靠他人，我们一家人可以搞定所有的事情。

talk is cheap

准备在公司写点什么，放假放的过于仓促，其实也不知道写一些什么。年终奖又一次没有发，其实我倒无所谓，就像承诺的奖金也不知道什么时候发，那虚无缥缈的四年的年终奖一起发也随着风飘散了吧。我其实并不在意这些东西，我只是在意一些承诺，一些言之凿凿的承诺罢了。

Fenng说的对，与其纠结这些，不如思考自己的将要要做什么，行业不会对真正有抱负的人有所亏欠。

so, talk is cheap, I need to build something.

17年我会做几个外包性质的项目，多为创业项目的0.1版本，然后做点自己的业余SAAS项目，积累一些自己的东西。我终究不会在一个公司呆很久，也不会去一个所谓的大公司甩尾。我讨厌大公司，只要有足够的利润养活自己，我宁愿做自己的事情，做自己喜欢的东西，也不愿为钱而一味的加班，透支自己的生命。

我需要一些时间来完成自己心中的想法，be patient!

build the furture

今年在深圳过年，一家三口也是其乐融融，且深圳天气非常好，过一个美好的春节也是可以预见的。

2016年发生很多不愉快的事情，就随着时间逝去吧。自己的未来要掌握在自己的手中，不要被那些不爽的情绪
所困扰。

有时候觉得自己过于多愁善感，这样实在不好。

这周开始读《战天京》，非常有意思。曾国藩三十岁到四十岁的时候，也没办法修炼到后来的境界，也是急急
忙忙，脾气暴躁。这个近代完人，浑身有那么多毛病，何况我这种人。修炼是长期的事情，就像跑步减肥一样，
没有一两年，是没办法见到成效的。人生不在于这一两年的困顿。没有谁的人生是一帆风顺的，最要紧的还是
平常心和持续去做，坚韧不拔的精神。
