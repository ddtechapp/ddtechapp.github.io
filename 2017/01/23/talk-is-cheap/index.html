talk is cheap

准备在公司写点什么，放假放的过于仓促，其实也不知道写一些什么。年终奖又一次没有发，其实我倒无所谓，就像承诺的奖金也不知道什么时候发，那虚无缥缈的四年的年终奖一起发也随着风飘散了吧。我其实并不在意这些东西，我只是在意一些承诺，一些言之凿凿的承诺罢了。

Fenng说的对，与其纠结这些，不如思考自己的将要要做什么，行业不会对真正有抱负的人有所亏欠。

so, talk is cheap, I need to build something.

17年我会做几个外包性质的项目，多为创业项目的0.1版本，然后做点自己的业余SAAS项目，积累一些自己的东西。我终究不会在一个公司呆很久，也不会去一个所谓的大公司甩尾。我讨厌大公司，只要有足够的利润养活自己，我宁愿做自己的事情，做自己喜欢的东西，也不愿为钱而一味的加班，透支自己的生命。

我需要一些时间来完成自己心中的想法，be patient!
