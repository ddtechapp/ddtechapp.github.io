Code Artist & Father

2017-01-04

start, szyupeng-tech.com

每年一月份，内心就开始骚动，各种想法层出不穷，可想之后，什么也没干，一年的蹉跎又开始了。该上班，该加班就加班，该刷剧刷剧。

2016年没有看一部完整的电影….

今天注册了域名，购买了腾讯云主机，搭建官网中，我要开始真正干活了。以前想到的种种创业的想法，都没有办法一一实现。创业绝对不是几个程序员，没日没夜的蜗居在黑暗的小房子里面不停的写代码就可以做成的。能够吃苦的人很多，能够成功的人还是那么少。吃苦只是最最基本的技能而已。

我是个普通人，普通的不能再普通的人，不高不帅也不酷，而且还有点小胖。就像村上春树说的那样，总是会被安排到最差的位子的那种人，放入人群不会被别人发现的那种人。可我依旧想开花，想结果，想做一些有意义的事情。

互联网那种创业已经不可避免的远离我去了，我不会再去追逐那些看似完美的梦幻，我要脚踏实地的做出点东西来，从最基础的事情做起-外包。

比如公众号开发，小程序开发，iOS/Android APP开发，网站开发，甚至帮助别人开发创业项目也行。只要是能够挣钱的活，我都应该去干。神马PS，神马SEO都要去做一做。不在乎微利不微利，不在乎技术是否高大上，空有一身技术，无法养活自己的家人是可耻的。

公司名字叫煜鹏网络，应该没人注册，立意第一来自我儿子的名字，昱芃，第二煜为明亮，鹏为展翅飞翔的大鸟，深圳也叫鹏城。这个名字非常满意。

目前是第一天，什么也别说，干起来吧！

中午开始读村上春树的小说-我的职业是小说家，刚看完第一篇，非常不错，二十岁到三十岁的人，都是拮据的，没什么不可以失去的。这本小说给我了很多的鼓励和慰藉。

写代码了！
